# HR-Attrition-Analysis

Hr attrition analysis dashboard and insights using PowerBI

# Project Overview
Employee attrition is a growing challenge for organizations, yet HR teams often lack a **clear & measurable understanding** of:
- How severe the attrition problem is
- Which employee groups are leaving the most
- What factors are driving employee exits

This project delivers an **interactive Power BI dashboard** supported by **documented insights and actionable recommendations** to help HR leadership make data-driven retention decisions.

##  Power BI Dashboard Structure

### Page 1: Interactive HR Attrition Dashboard
This page provides a high-level and detailed analytical view of attrition using:
- KPI cards for workforce health
- Attrition analysis by:
  - Education Field
  - Age Group
  - Job Role
  - Salary Slab
  - Years at Company
  - Gender & Job Satisfaction
- Department-level slicers for interactive filtering

----

### Page 2: Insights, Problems & Actions (Summary)
This page contains:
- business insights
- Identified attrition problems
- recommended HR actions
